FAQ

A companion checklist for a Dark Souls 3 playthrough — track bosses, missable items, NPC quests, rings, spells, upgrades, and gear as you go, with your progress saved automatically in your browser. It works offline and can be installed like an app.

This project is built on Zachary Kjellberg's Dark Souls 3 Cheat Sheet, which this repository was originally forked from. That in turn adapted the source code of Stephen McNabb's Dark Souls 2 Cheat Sheet, and the walkthrough is based on DeathGodGarra's NPC Side Quests Guide V2. It has evolved thanks to the many GitHub contributors with great suggestions, fixes, and improvements!

Is this a guide?

Nope, not really. This is a set of checklists and information that you can use while playing Dark Souls to make sure you don't miss an item, action, conversation, or boss. If you need playthroughs then you can click on the zone titles to read the wiki walkthrough.

Do I need to follow the playthrough in this exact order?

Dark Souls 3 is not a linear game and has multiple ways of progressing. This guide highlights zones as they become available, so feel free to follow however you please. You can always check things off in a different order.

How do the filters and NG / NG+ / NG++ buttons work?

On the Playthrough tab, the filter toolbar lets you hide whole categories of entries (bosses, missables, rings, spells, gear, materials, and more) so you can focus on just what you care about — click a category or item type to toggle it. The Builds group hides everything tied to a caster build you're not playing — its catalysts (staves, pyromancy flames, talismans and chimes), rings, trainers, and spells. Filters only affect the Playthrough: the Achievements and Weapons/Shields tabs are 100% completion lists, so nothing is ever hidden there. The NG / NG+ / NG++ buttons switch which journey you're on: some entries only show up on NG+ and beyond (such as +1 and +2 rings), while a few disappear on NG+ (such as the Ashen Estus Flask), so the list always reflects the journey you selected. The Hide Completed toggle hides everything you've already checked off.

What do the Sorcery / Pyromancy / Miracle buttons do?

They highlight, rather than hide: toggle one (or several) on any checklist tab and every entry connected to that caster build gets a colored tint — the spells, the catalysts that cast them (staves, pyromancy flames, talismans and sacred chimes), build-related rings, and the merchants who teach the spells. An entry shared by more than one active build (Karla, for example) shows a blend of the colors. The highlight is a scratch view for planning a build — it isn't saved and resets on reload.

What are the star buttons next to some equipment?

Equipment with a unique passive effect — the Grass Crest Shield's stamina regen, catalysts with unusual scaling, and so on — gets a small button in the Weapons/Shields and Armor tabs. Click it to read the effect without leaving the page (it never toggles the checkbox). While Sorcery / Pyromancy / Miracle highlights are active, a description that matters to your build is tinted in the same colors and the spell schools it names are marked — including cross-school effects like the Saint-tree Bellvine, whose casting speed buff applies to every school even from the off-hand.

Can I use this for multiple characters?

Yup, use the profile selector and buttons in the Options tab at the top of the page to set up multiple profiles, each with its own independent progress.

How does my progress get saved?

Your progress is saved automatically in your browser's local storage, separately for each profile — there's no account and nothing is sent to a server. Because it lives in the browser, clearing your browsing data or site data will erase it, and private/incognito windows won't keep it. To be safe, export a backup from the Options tab (see below).

Can I move my progress to another device or browser?

Yes. In the Options tab, use Export file to download all of your profiles as a .json file, then Import file on the other device to load it. There's also Export clipboard / Import textbox to move data by copy-paste. This is also the recommended way to keep backups. Note that importing replaces your current profile data.

Does it work offline? Can I install it?

Yes — it's a Progressive Web App. Once you've opened it online at least once, it keeps working with no network connection. Most browsers also let you install it: use "Add to Home Screen" on mobile, or the install icon in the address bar on desktop, to launch it like a native app.

I have feedback, how can I contribute?

You can visit the GitHub repository and report Issues or create a fork and submit a Pull Request.
